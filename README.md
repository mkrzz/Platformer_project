# Eyedrop

### University assignnment 
Create your own platform game using the SFML API  
This repository is a portfolio version of the original coursework, all features and content implemented by myself

## Theme  
A continuous vertical scroller platform game where you control an eyeball.  
The objective is to collect all 20 moisture orbs while navigating platforms and avoiding enemies.  

## Features  
- Collect all 20 moisture orbs to win
- Jump between platforms in a verctical-scrolling environment
- Avoid spikey enemy arrows
- Gravity mechanic: Staying on platforms too long causes you to sink through
- Health mechanic

## Libraries 
SFML  

## How to play  
- Compile and run the executable
- Use A and D to move left and right, with spacebar to jump

## Assets / credits  
- 2D images (platforms, orbs, eyeball, arrows) downloaded from Kenney.nl
- Vector2 files provided by the University
- All other code and game logic implemented by me
- All sound files downloaded from Soundsnap
